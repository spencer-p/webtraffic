Traffic Simulation

Welcome to the traffic simulation web portal. You may want to bookmark this home page since there are no navigational links on the other pages. Sorry about that.

If you don't know what this is, you probably shouldn't be here. Alternatively you can take a look at this repository for some more info.

A note to simulation runners - I didn't make it very easy to distinguish between inputs. Every iteration gets essentially a random number assigned to it (a hash of the input string). Since there is no title or helpful metadata, I suggest you add an unused "Note" or "Title" field to the input JSON like so:

{ "Note": "This simulation is testing whether or not cars == good.", "BusRouteInfo": [ ... ], "CarRouteInfo": [ ... ], ... }

Remember the comma after the extra line! Note that the "Note" string can be any text, as long as it does not collide with the other keys. Hope that helps.
